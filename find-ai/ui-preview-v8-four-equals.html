What's different in v8
01 · Chat is co-equal
4×1 row, not 2×2 bento

Chat is tile 04 with the same card, same size, same icon treatment. The "3 tools + chat" mental model dissolves. Every tile is one verb ("Can I trust...?") + one output (PDF or WhatsApp copy).

02 · 4 scroll acts
Momentum, not a poster

Hero → 4 Tools → PDF Proof → Legal Backbone. Each act is 100svh/dvh so every swipe reveals a new aha. No dashboard bloat on landing (that moves to /profile).

03 · Thumb-zone sticky CTA
Always bottom, always safe

"Open the toolkit →" stays pinned with backdrop-blur + env(safe-area-inset-bottom). One-handed pro use, CLAUDE.md doctrine compliant.

The two-surface split
Landing = feature showcase. Profile = personal dashboard.

This page (v8) does ONE job: show the four tools as equals, earn the click. The dashboard-style surface (v5d soft bento) moves to /profile — user's open cases, saved PDFs, trust grade. Two pages, two jobs, zero mixing. Linear / Notion / Stripe all structure entry flows this way.
