Zeus's pick
v5 Apple Minimal.

For Phase 1 public launch: v5. Three reasons. (1) The 2×2 tool bento fits one mobile screen — compliance tools need to be immediately scannable, not scrolled. (2) Gradient icons per tool = memorability without committing to one brand color. (3) It reads as "serious but not scary" — which is exactly what a first-time landlord needs before handing over their keys.

Keep v6 Pastel in the back pocket as a possible "education / explainer" variant later — it's warmer and better for blog posts + onboarding walkthroughs.

Skip v7 Mono for the primary landing — it's stunning design work but in a MY market still warming up to AI tools, pure B&W can read as cold/technical. Park it for an "enterprise" or "for agencies" page variant.
