# BLAST TEMPLATES

Standard WhatsApp blast templates for landlord outreach. Always use these as the base. Personalise with landlord name and unit number before sending. Never modify the sign-off.

If your unit is available for rent, I would love to help you find a good tenant quickly. Please let me know the details — rental price, furnishing, and available date. Thank you.

## Rules for all blast messages

- Always personalise with landlord name and unit number.
- Never use repeated identical emojis.
- Never mention "broadcast list", "WhatsApp list", or "add you to my list".
- Always include sign-off: Ken Tan | Gather Properties | REN 31548 | 016-713 5601
- Follow-up messages must be approved by Mr Ken before sending.
- Malay template for Malay landlords only — check name before sending.
