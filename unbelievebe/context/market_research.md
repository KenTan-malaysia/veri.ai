# Market Research

Load this file when: market, research, area overview, pricing, competitive analysis, what's available in [area].

---

## What to Deliver

When Ken asks about a specific area, produce a full market overview covering all project types:
- Condominiums / Apartments
- Landed (terraced, semi-D, bungalow)
- Shoplots / Commercial
- Industrial parks / Warehouses

## Required Columns (Excel-friendly output)

| Column | Description |
|---|---|
| Project Name | Full name of the development |
| Property Type | Condo / Landed / Shoplot / Industrial |
| Location | Area, state |
| Developer | Developer name |
| Price Range | Approximate asking price or rental range (RM) |
| Status | Under construction / Completed / Subsale |
| Notes | Anything relevant — tenure, target market, vacancy rate |

## Coverage Areas

KL · Selangor · Penang · Perak

Never produce a market overview that misses an active project in Ken's coverage area.

## Format

Default: Excel-friendly table (no merged cells, sortable).
If Ken asks for summary only: WhatsApp-ready plain text overview.

## Template File

Market_Research_Template.xlsx is in `context/` — use it as the base when producing market data.

---

## Three-Sheet Structure

**Sheet 1 — Market Overview**
All active projects in the area. Columns above.

**Sheet 2 — Competitor Listings**
What other agents or landlords are listing — price, unit type, furnishing, availability.

**Sheet 3 — Summary**
Key takeaways: price range, demand level, gaps Ken can exploit, recommended positioning.
